## 🚀 מסלול Admin ראשון — מ-0 עד בניין פעיל

**צעד 1 — הרשמה** (2 דקות)
→ index.html → הרשמה עם אימייל + שם + שם בניין + דמי ועד

**צעד 2 — הגדרת בניין** (3 דקות)
→ Settings: וידוא שם בניין ודמי ועד נכונים

**צעד 3 — הזמנת דיירים** (5 דקות)
→ Tenants → "הזמן דייר" → שלח קישור לכל דייר
→ או: הוסף ידנית עם "הוסף דייר"

**צעד 4 — שליחת הודעה ראשונה** (1 דקה)
→ Messages → כתוב הודעה → "כל הדיירים" → שלח

**צעד 5 — רשום תשלום ראשון** (2 דקות)
→ Finance → "רשום תשלום" → בחר דייר → סכום → שמור

**סה"כ: ~13 דקות עד בניין פעיל עם דיירים, הודעות ותשלומים.**
